# Customer Journey & Funnel Analysis Dashboard

## 📌 Project Objective
To create a Power BI dashboard that tracks customer behavior from awareness to purchase, analyzing conversion rates at each funnel stage and identifying drop-off points.

## 📊 Business Problem
E-commerce and marketing teams need to understand where customers drop off in the buying journey. This dashboard helps identify bottlenecks, optimize acquisition channels, and improve customer retention.

## 📁 Dataset Description
- **Source:** Generated realistic dataset (200 customers)
- **Columns:** Customer ID, Customer Segment, Region, Acquisition Channel, Product Viewed, Add to Cart, Checkout Started, Purchase Completed, Purchase Value, Customer Retention Status, etc.

## 🛠️ Tools Used
- Power BI Desktop (DAX, Power Query)
- GitHub

## 📈 Key Metrics (KPIs)
| KPI | Value |
|-----|-------|
| Total Visitors | 200 |
| Leads Generated | 154 |
| Conversion Rate | 15% |
| Add-to-Cart Rate | 34% |
| Checkout Rate | 24% |
| Total Revenue | ₹664.5K |
| Retention Rate | 79.5% |

## 📊 Dashboard Features
- **KPI Cards** – 7 top-level metrics
- **Funnel Chart** – Stage-wise customer count (200 → 154 → 68 → 48 → 30)
- **Channel Performance** – Acquisition channel vs conversion
- **Segment-wise Conversion** – Customer segment analysis
- **Regional Conversion** – Region-wise performance
- **Customer Retention** – Donut chart (Active 51%, At Risk 28.5%, Churned 20.5%)
- **Drop-off Analysis Table** – Stage-wise drop-off percentages
- **Recommendations Text Box** – Actionable insights
- **Slicers** – Region, Acquisition Channel, Age Group, Customer Segment

## 🔍 Key Insights
- **Biggest bottleneck:** Product Viewed → Add to Cart (55.8% drop-off)
- **Second bottleneck:** Checkout → Purchase (37.5% drop-off)
- **Best performing channel:** Organic Search (7 conversions)
- **Best performing campaign:** Winter Sale (₹163.6K revenue)
- **Highest conversion region:** West (17.5%)
- **Retention rate:** 79.5% (Active 51%, At Risk 28.5%, Churned 20.5%)

## ✅ Recommendations
- Optimize product pages (CTAs, reviews, stock visibility)
- Simplify checkout process (guest checkout, multiple payments)
- Retarget cart abandoners with email/SMS
- Increase budget on Organic Search
- Scale Winter Sale-style campaigns
- Re-engage At Risk customers (15% discount)
- Win-back Churned customers (30% off + free shipping)
